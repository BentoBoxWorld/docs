# Stranger Realms Permissions

| Permission | Description | Default |
| --- | --- | --- |
| `strangerrealms.island` | Allow claim command usage | `true` |
| `strangerrealms.claim` | Allow claim command usage | `true` |
| `strangerrealms.unclaim` | Allow unclaim command usage | `true` |
| `strangerrealms.create` | Allow claim creation | `false` |
| `strangerrealms.island.home` | Allow teleporting to player claim | `true` |
| `strangerrealms.island.sethome` | Let the player use the sethome command | `true` |
| `strangerrealms.island.info` | Let the player check other players info | `true` |
| `strangerrealms.island.lock` | Allows claim locking | `true` |
| `strangerrealms.island.near` | Players can see nearby claim names | `true` |
| `strangerrealms.island.expel` | Allows expelling of visitors | `true` |
| `strangerrealms.island.ban` | Allows banning of visitors | `true` |
| `strangerrealms.island.settings` | Player can see server settings | `true` |
| `strangerrealms.island.language` | Player can select a language | `true` |
| `strangerrealms.island.name` | Player can set the name of their claim | `true` |
| `strangerrealms.spawn` | Player can use the claim spawn command if spawn exists | `true` |
| `strangerrealms.island.reset` | Player can use the claim reset or restart command | `false` |
| `strangerrealms.island.team` | Let a player use team command | `true` |
| `strangerrealms.island.team.setowner` | Let a player change the team owner | `true` |
| `strangerrealms.island.team.invite` | Let a player invite others | `true` |
| `strangerrealms.island.team.reject` | Let a player reject invites | `true` |
| `strangerrealms.island.team.leave` | Let a player leave the team | `true` |
| `strangerrealms.island.team.kick` | Let a player kick team members | `true` |
| `strangerrealms.island.team.accept` | Let a player accept invitations | `true` |
| `strangerrealms.island.team.trust` | Let a player use team trust commands | `true` |
| `strangerrealms.island.team.coop` | Let a player use team coop commands | `true` |
| `strangerrealms.island.team.promote` | Let a player use promote commands | `true` |
| `strangerrealms.island.settings.*` | Allow use of settings on claim | `true` |
| `strangerrealms.mod.info` | Let a moderator see info on a player | `op` |
| `strangerrealms.mod.clearreset` | Allow clearing of claim reset limit | `false` |
| `strangerrealms.mod.bypasscooldowns` | Allow moderator to bypass cooldowns | `op` |
| `strangerrealms.mod.bypassdelays` | Allow moderator to bypass delays | `op` |
| `strangerrealms.mod.bypassprotect` | Allow moderator to bypass claim protection | `op` |
| `strangerrealms.mod.bypassexpel` | Allow moderator to bypass claim expulsion | `op` |
| `strangerrealms.mod.switch` | Allows moderator to switch bypass protection on and off | `op` |
| `strangerrealms.mod.lock` | Locks or unlocks an claim | `op` |
| `strangerrealms.mod.bypasslock` | Bypasses an claim lock | `op` |
| `strangerrealms.mod.bypassban` | Bypasses claim ban | `op` |
| `strangerrealms.mod.team` | Enables modification of teams via kick and add commands | `false` |
| `strangerrealms.admin.tp` | Allows teleport to an claim | `op` |
| `strangerrealms.admin.clearresetall` | Allow clearing of claim reset limit of all players | `op` |
| `strangerrealms.admin.reload` | Reload the config.yml | `op` |
| `strangerrealms.admin.delete` | Let a player completely remove a player (including claim) | `op` |
| `strangerrealms.admin.register` | Let a player register the nearest claim to another player. | `op` |
| `strangerrealms.admin.unregister` | Removes a player from an claim without deleting the claim blocks. | `op` |
| `strangerrealms.admin.setspawn` | Allows use of spawn tools | `op` |
| `strangerrealms.admin.setrange` | Allows setting of claim protection range | `op` |
| `strangerrealms.admin.settingsreset` | Resets all the claims to default protection settings | `op` |
| `strangerrealms.admin.noban` | Player cannot be banned from an claim | `op` |
| `strangerrealms.admin.noexpel` | Player cannot be expelled from an claim | `op` |
| `strangerrealms.admin.setlanguage` | Resets all player languages and sets the default language | `op` |
| `strangerrealms.admin.getrank` | Get a player's rank | `op` |
| `strangerrealms.admin.setrank` | Set a player's rank | `op` |
